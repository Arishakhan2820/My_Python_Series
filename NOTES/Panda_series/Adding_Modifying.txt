📘 Pandas Topic 6: Adding $ Modifying Columns

What are Column Operations?

Column Operations are the actions performed on the columns of a DataFrame, such as adding new columns, modifying existing columns, deleting unnecessary columns, or renaming column names.

These operations help organize, clean, and prepare data for analysis and Machine Learning.

Definition

Column Operations in Pandas are the process of creating, updating, deleting, and renaming columns in a DataFrame. They help transform raw data into a more meaningful and organized format.

Why Do We Use Column Operations?

Column operations are used to:

Add new information to a dataset.
Perform calculations using existing data.
Create categories based on conditions.
Remove unnecessary columns.
Rename columns to make them more meaningful.
Prepare data for Data Analysis and Machine Learning.
Types of Column Operations

There are five common column operations in Pandas:

Adding a New Column
Creating a Column Using Calculations
Creating a Column Using Conditions
Deleting a Column
Renaming a Column
1. Adding a New Column

A new column can be added to a DataFrame by assigning values to a new column name.

The same value can be assigned to every row, or different values can be assigned individually.

Example

Adding a Passed column where every student has the value True.

Real-Life Example

A school has a student database.

The principal decides to add a new column called Attendance Verified.

Initially, every student is marked as True, indicating that their attendance has been verified.

2. Creating a Column Using Calculations

A new column can also be created by performing mathematical operations on existing columns.

Pandas automatically performs the calculation for every row without using loops.

Example

Creating a Marks_Out_50 column by dividing the marks by 2.

Real-Life Example

A school conducts an exam out of 100 marks, but another report requires marks out of 50.

Instead of calculating each student's marks manually, Pandas automatically converts every student's marks into the new scale.

3. Creating a Column Using Conditions

Sometimes a new column depends on certain rules or conditions.

Pandas allows us to assign values based on conditions, such as grades, pass/fail status, or categories.

This is commonly done using .apply() with a function.

Example

Assigning grades based on marks:

Marks 90 or above → Grade A
Marks 80–89 → Grade B
Marks Below 80 → Grade C
Real-Life Example

A teacher prepares report cards.

Instead of assigning grades manually to every student, Pandas automatically checks each student's marks and assigns the correct grade.

Understanding .apply()

The .apply() method applies the same function or rule to every value in a column.

It processes one value at a time until every row has been checked.

Why is .apply() Useful?
Avoids writing loops.
Applies the same rule to every row.
Makes the code shorter and cleaner.
Saves time when working with large datasets.
Understanding lambda

A lambda function is a small anonymous function used for simple tasks.

It is commonly used with .apply() when the logic is short.

Instead of writing a separate function, the rule can be written directly.

Real-Life Example

Imagine a teacher follows one grading rule for every student.

Instead of checking each student manually, the teacher applies the same grading rule to everyone.

That is exactly what .apply() and lambda do together.

4. Deleting a Column

Sometimes a dataset contains unnecessary columns.

These columns can be removed to make the dataset cleaner and easier to analyze.

Example

Removing the Marks_Out_50 column after it is no longer needed.

Real-Life Example

A company removes an old Temporary ID column from its employee database because it is no longer useful.

Understanding axis

Many Pandas functions use the axis parameter.

It tells Pandas whether the operation should be performed on rows or columns.

Axis	Meaning
axis = 0	Rows
axis = 1	Columns
Easy Trick
0 → Up and Down → Rows
1 → Left and Right → Columns
Understanding inplace=True

Normally, many Pandas functions create a new DataFrame after making changes.

When inplace=True is used, the changes are made directly to the original DataFrame.

Why Use inplace=True?
Saves memory.
No need to create another DataFrame.
Keeps the original DataFrame updated.
5. Renaming a Column

Sometimes column names are too short, unclear, or inconsistent.

Pandas allows us to rename columns with more meaningful names.

Example

Changing the column name from Marks to Total_Marks.

Real-Life Example

A company changes a column named Emp_ID to Employee_ID so that the report is easier to understand.

Viewing Column Names

Pandas provides a simple way to display all column names in a DataFrame.

This helps verify whether columns have been added, removed, or renamed successfully.
