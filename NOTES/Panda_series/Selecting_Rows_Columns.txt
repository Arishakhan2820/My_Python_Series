📘 Pandas Topic 4: Selecting Rows and Columns
What is Data Selection?

Data Selection is the process of choosing specific rows or columns from a DataFrame.

Instead of using the entire dataset, we often work with only the data we need. Pandas provides simple and powerful methods to select columns, rows, or a combination of both.

Definition

Data Selection in Pandas is the process of retrieving specific rows, columns, or subsets of data from a DataFrame for analysis, filtering, or processing.

Why Do We Select Data?

Data selection is used to:

View only the required information.
Work with specific columns.
Access individual records.
Analyze selected data.
Improve code readability.
Process large datasets more efficiently.
Types of Data Selection

There are three common ways to select data in a DataFrame:

Selecting Columns
Selecting Rows
Selecting Rows and Columns Together
1. Selecting Columns

Columns contain one type of information, such as:

Name
Age
City
Marks
Grade

Sometimes we only need one column, while other times we need multiple columns.

Selecting a Single Column

Selecting one column returns a Series.

Since only one column is selected, the output becomes a one-dimensional (1D) data structure.

Example

Selecting the Name column returns only the list of student names.

Selecting Multiple Columns

Selecting multiple columns returns a DataFrame.

Because multiple columns form a table, the output remains a two-dimensional (2D) DataFrame.

Example

Selecting Name, Marks, and Grade returns a table containing only these three columns.

Difference Between Single and Multiple Column Selection
Single Column	Multiple Columns
Returns a Series	Returns a DataFrame
One-dimensional (1D)	Two-dimensional (2D)
Contains one column	Contains multiple columns
2. Selecting Rows

Rows represent individual records in a DataFrame.

Pandas provides two main methods to select rows:

iloc[]
loc[]

Both methods perform similar tasks but use different ways to identify rows.

iloc[] (Integer Location)
What is iloc[]?

iloc[] selects data using integer positions (index numbers).

It works with the row and column numbers, not their names.

Features of iloc[]
Uses numeric indexes.
Supports slicing.
Supports selecting rows and columns together.
The ending index is not included when slicing.
Example Uses
Select the first row.
Select the last row.
Select rows between two positions.
Select specific rows and columns using numbers.
Real-World Example

Imagine students are standing in a line.

Instead of calling them by name, the teacher says:

Student number 1
Student number 2
Student number 3

This is similar to iloc[], where selection is based on position.

loc[] (Label Location)
What is loc[]?

loc[] selects data using labels, such as index labels or column names.

Instead of positions, it uses meaningful names.

Features of loc[]
Uses labels instead of positions.
Can select rows using index labels.
Can select columns using column names.
The ending label is included when slicing.
Example Uses
Select rows by index labels.
Select columns by their names.
Select both rows and columns together.
Real-World Example

Imagine a teacher calls students by their names instead of roll numbers.

The teacher says:

Ali
Sara
Fatima

This is similar to loc[], where selection is based on labels.

Selecting Rows and Columns Together

Pandas also allows selecting both rows and columns at the same time.

This is useful when only a specific part of a dataset is required.

For example:

First three students
Only Name and Marks columns

Instead of selecting the whole table, only the required portion is returned.

Difference Between iloc[] and loc[]
iloc[]	loc[]
Uses integer positions.	Uses labels or names.
Works with row and column numbers.	Works with row labels and column names.
End index is not included in slicing.	End label is included in slicing.
Position-based selection.	Label-based selection.

Key Points to Remember
A single selected column returns a Series.
Multiple selected columns return a DataFrame.
iloc[] selects data using integer positions.
loc[] selects data using labels.
iloc[] excludes the ending index during slicing.
loc[] includes the ending label during slicing.
Both methods can select rows and columns together.

🎯 Real-Life Example

Imagine a school has a student database containing 5,000 students.

The principal wants to:

View only the Name column.
Check Name and Marks together.
Display the first 10 student records.
View students from row 100 to row 200.
Display only Name, Age, and Marks for selected students.

Instead of searching the entire database manually, Pandas allows quick selection using column selection, iloc[], and loc[].
