📘 Pandas Topic 5: Filtering Data
What is Filtering?

Filtering is the process of selecting only the rows that satisfy a specific condition.

Instead of viewing the entire dataset, filtering allows us to display only the information that matches our requirements.

For example, instead of showing all students, we can display only students who scored more than 80 marks.

Definition

Filtering in Pandas is the process of selecting rows from a DataFrame based on one or more conditions. It helps extract only the required data while ignoring the remaining records.

Why Do We Use Filtering?

Filtering is used to:

Find specific records quickly.
Analyze only the required data.
Remove unnecessary information.
Prepare datasets for analysis.
Make data easier to understand.
Work efficiently with large datasets.
How Does Filtering Work?

Filtering checks every row in a DataFrame against a given condition.

If the condition is True, the row is displayed.
If the condition is False, the row is ignored.

Only the rows that satisfy the condition appear in the result.

Types of Filtering

Filtering can be performed using different types of conditions.

1. Comparison Filtering

Comparison operators compare values with a condition.

Common comparison operators include:

Operator	Meaning
==	Equal to
!=	Not equal to
>	Greater than
<	Less than
>=	Greater than or equal to
<=	Less than or equal to
Example Conditions
Marks greater than 80.
Age less than 21.
Grade equal to "A".
City equal to "Karachi".
2. Multiple Conditions

Sometimes one condition is not enough.

We may need to apply two or more conditions together.

Pandas supports logical operators for this purpose.

Logical Operators
Operator	Meaning
&	AND (both conditions must be True)
`	`
~	NOT (reverses the condition)
Example

Find students who:

Live in Karachi AND
Have marks above 80.

Both conditions must be true.

Another example:

Show students who are:

From Lahore OR
Have Grade A.

Only one condition needs to be true.

Boolean Mask

Filtering creates a Boolean Mask behind the scenes.

A Boolean Mask is a list containing only:

True
False

For each row:

True → Keep the row.
False → Remove the row.

This is how Pandas decides which rows to display.

Why is Filtering Important in Machine Learning?

Before training a Machine Learning model, datasets often contain:

Unnecessary rows.
Invalid records.
Incorrect values.
Outliers.

Filtering helps remove unwanted records so that only useful data is used for training the model.

This improves the quality and accuracy of Machine Learning models.
