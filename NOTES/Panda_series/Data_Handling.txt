📘 Pandas Topic 7: Handling Missing Data (Real-World Data Cleaning)

What is Missing Data?

Missing data refers to information that is not available or has not been recorded in a dataset.

In Pandas, missing values are usually represented as:

NaN (Not a Number) → Used for missing numerical values.
None → Used for missing text (string) values.

Almost every real-world dataset contains some missing values, so handling them correctly is an important part of Data Analysis and Machine Learning.

Definition

Handling Missing Data is the process of identifying, analyzing, removing, or replacing missing values in a dataset so that the data becomes clean, complete, and ready for analysis.

Why Do We Handle Missing Data?

Missing values can cause problems such as:

Incorrect calculations.
Wrong analysis results.
Poor Machine Learning model performance.
Errors during data processing.
Incomplete reports.

Cleaning missing data improves the quality and accuracy of the dataset.

Types of Missing Values
1. NaN (Not a Number)

NaN represents missing numerical data.

It is commonly used for columns containing numbers such as age, salary, marks, or price.

Example
Name	Age
Ali	20
Sara	NaN
Usman	21

Sara's age is missing, so Pandas stores it as NaN.

2. None

None represents missing text or object values.

It is commonly used in string columns.

Example
Name	City
Ali	Karachi
Sara	Lahore
Usman	None

Usman's city is unknown, so Pandas stores it as None.

How to Find Missing Data?

Pandas provides functions to detect missing values.

1. isnull()

The isnull() function checks every cell in the DataFrame.

True → Value is missing.
False → Value is available.

This helps identify exactly where data is missing.

Real-Life Example

Imagine a teacher checks every student's form.

Whenever information is missing, the teacher marks that field.

Similarly, isnull() marks missing values with True.

2. isnull().sum()

After finding missing values, we often need to know how many are missing in each column.

The sum() function counts all the True values.

This gives the total number of missing values in every column.

Real-Life Example

A school principal wants to know:

How many students have missing ages?
How many students have missing marks?

Instead of checking manually, Pandas counts them automatically.

How to Handle Missing Data?

There are two common methods.

Method 1: Remove Missing Data (dropna())

The dropna() function removes rows or columns that contain missing values.

This method is useful when only a small amount of data is missing and removing it will not affect the overall dataset.

Real-Life Example

A company receives 1,000 customer forms.

Only 5 forms are incomplete.

Instead of correcting them manually, the company simply removes those incomplete forms.

Method 2: Fill Missing Data (fillna())

Instead of deleting data, missing values can be replaced with another value.

This keeps the dataset complete.

Filling Numeric Values

For numerical columns, missing values are commonly replaced with:

Mean (Average)
Median
Mode
Any fixed number
Mean (Average)

The mean is calculated by adding all available values and dividing by the total number of available values.

It represents the average value of the column.

Real-Life Example

Suppose three students have ages:

20, 21, and 20.

The average age is 20.33.

If another student's age is missing, it can be replaced with 20.33.

This keeps the dataset balanced.

Filling Text Values

For text columns, missing values are usually replaced with words such as:

Unknown
Not Available
Missing
N/A
Real-Life Example

If a customer's city is not known, instead of leaving it empty, we can write Unknown.

This makes the dataset easier to understand.

Understanding inplace=True

Many Pandas functions return a new DataFrame after making changes.

When inplace=True is used, the changes are made directly to the original DataFrame.

This means no new DataFrame is created.

Why Use inplace=True?
Saves memory.
Updates the original dataset.
Makes code shorter and simpler.

Why is Missing Data Important in Machine Learning?

Machine Learning models cannot always work properly with incomplete datasets.

Before training a model, Data Scientists usually:

Find missing values.
Count missing values.
Remove unnecessary incomplete records.
Replace missing values with meaningful data.

This process is called Data Cleaning or Data Preprocessing.

Imagine you are working as a Data Analyst in a hospital.

The patient database contains:

Patient	Age	Blood Group	City

Ahmed	25	NaN	None

Before creating reports or training a Machine Learning model, you:

Find missing values.
