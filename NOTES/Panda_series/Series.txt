📘 Pandas Topic 1: Series

What is a Series?

A Series is a one-dimensional (1D) data structure in Pandas that stores a single column of data. It contains two parts:

Index → The label or position of each value.
Value → The actual data stored in the Series.

Think of a Series as a single column in a table.

Definition

A Pandas Series is a one-dimensional labeled array that can store different types of data, such as numbers, text, or Boolean values. Each value in a Series has an associated index, making it easy to access and manage data.

Why Do We Use a Series?

A Series is used to:

Store a single column of data.
Organize data with labels (indexes).
Perform mathematical calculations easily.
Filter and analyze data quickly.
Access values using indexes or custom labels.
Key Components of a Series

Every Series has two main parts:

1. Index

The index is the label or position of each value.

If you do not provide an index, Pandas automatically creates one starting from 0.

Example:

Index	Value
0	85
1	90
2	78
3	92
4	88
2. Values

The values are the actual data stored in the Series.

Example:

85
90
78
92
88
Default Index

If no index is provided, Pandas automatically assigns numeric indexes:

0
1
2
3
4

This is called the default index.

Custom Index

Instead of using numbers, you can create your own labels as indexes.

Example:

Ali
Sara
Usman
Fatima
Bilal

Using custom indexes makes the data easier to understand and allows you to access values by name instead of position.

Accessing Data

A value in a Series can be accessed by using:

The default numeric index.
A custom label (such as a student's name).

This makes data retrieval simple and readable.

Series Properties
1. Values

Returns only the data stored inside the Series.

Example:

85
90
78
92
88
2. Index

Returns all index labels.

Example:

Ali
Sara
Usman
Fatima
Bilal
3. Data Type (dtype)

Shows the type of data stored in the Series.

Common data types include:

int64 → Integer numbers
float64 → Decimal numbers
object → Text (strings)
bool → True or False values
4. Shape

Returns the size of the Series.

Example:

(5,)

This means the Series contains 5 values.

Mathematical Operations

Pandas allows mathematical calculations directly on a Series.

Common operations include:

Mean

Returns the average of all numerical values.

Maximum

Returns the largest value.

Minimum

Returns the smallest value.

Sum

Adds all values together.

These operations help summarize and analyze data quickly.

Filtering Data

Filtering allows you to display only the values that satisfy a specific condition.

For example, you can show:

Students with marks greater than or equal to 88.
Employees earning more than a certain salary.
Products costing above a certain price.

Filtering is one of the most useful features in Pandas because it helps you focus only on the required data.
