📘 Pandas Topic 2: DataFrame
What is a DataFrame?

A DataFrame is a two-dimensional (2D) data structure in Pandas that stores data in the form of rows and columns, just like an Excel spreadsheet or a database table.

A DataFrame is the most commonly used data structure in Pandas because it can store multiple columns of different types of data together.

Definition

A Pandas DataFrame is a two-dimensional labeled data structure that organizes data into rows and columns. Each column can store a different type of data, such as numbers, text, or Boolean values, making it ideal for handling real-world datasets.

Why Do We Use a DataFrame?

A DataFrame is used to:

Store complete tables of data.
Organize information in rows and columns.
Analyze and manipulate datasets easily.
Perform calculations on large amounts of data.
Prepare data for Data Science, Machine Learning, and Artificial Intelligence projects.
Key Components of a DataFrame

A DataFrame consists of four main components:

1. Rows

A row represents a single record or observation.

Example:

Name	Age	City	Marks
Ali	20	Karachi	85

This entire horizontal line is one row.

2. Columns

A column represents one type of information.

Example:

Name
Ali
Sara
Usman

Here, Name is a column.

Other examples of columns are:

Age
City
Marks
Passed
3. Index

The index identifies each row.

If no index is provided, Pandas automatically creates one starting from 0.

Example:

Index	Name
0	Ali
1	Sara
2	Usman
4. Values

Values are the actual data stored inside the DataFrame.

Example:

Ali
20
Karachi
85
True
Creating a DataFrame

One of the easiest ways to create a DataFrame is by using a Python dictionary.

In the dictionary:

Each key becomes a column name.
Each list of values becomes the data for that column.

For example:

"Name" becomes the Name column.
"Age" becomes the Age column.
"City" becomes the City column.
"Marks" becomes the Marks column.
"Passed" becomes the Passed column.

Pandas combines all these columns into one complete table called a DataFrame.

Dictionary to DataFrame

When a dictionary is converted into a DataFrame:

Every key becomes a column.
Every value in the list becomes a row entry.
Pandas automatically creates indexes if they are not provided.

This makes it very easy to organize structured data.

Relationship Between Rows and Columns

A DataFrame is made up of:

Multiple rows
Multiple columns

Each row contains information about one record.

Each column stores one category of information.

Data Types in a DataFrame

Different columns can store different types of data.

Common data types include:

int64 → Integer numbers
float64 → Decimal numbers
object → Text (strings)
bool → True or False values

Example:

Column	Data Type
Name	object
Age	int64
City	object
Marks	int64
Passed	bool

Difference Between Series and DataFrame
Series	DataFrame
One-dimensional (1D).	Two-dimensional (2D).
Contains one column of data.	Contains multiple columns of data.
Stores a single type of information.	Stores complete tables of information.
Simpler structure.	More powerful and flexible.
