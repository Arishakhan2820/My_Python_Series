TOPIC 2 — Building a Neural Network with TensorFlow (Student Pass/Fail Prediction)

 What is a Neural Network?

A Neural Network is a machine learning model inspired by the human brain. It learns patterns from data and uses those patterns to make predictions or decisions.

Instead of following fixed rules, a neural network learns from examples.

Real-World Example

Think of a teacher checking whether a student will pass or fail.

The teacher considers:

Study hours
Sleep
Attendance
Previous marks

After looking at many students, the teacher starts predicting whether a new student will pass.

A Neural Network works in the same way—it learns from previous data and predicts outcomes for new data.

🎯 Project Goal

The goal of this project is to build an AI model that predicts whether a student will:

Pass
Fail

The prediction is based on four features:

Study Hours
Sleep Hours
Attendance
Previous Marks

📊 Dataset

A dataset is simply a collection of information.

Each student has four pieces of information (features) and one result (label).

Features (Input)

These are the values the model uses to make predictions.

Study Hours
Sleep Hours
Attendance
Previous Marks
Label (Output)

This is the correct answer that the model tries to learn.

Pass = 1
Fail = 0
Real-World Example

Imagine a school report.

Study	Sleep	Attendance	Previous Marks	Result
8 hrs	7 hrs	90%	85	Pass
2 hrs	5 hrs	60%	50	Fail

The model studies many such examples to learn patterns.

 Splitting the Dataset:

Before training the model, the data is divided into two parts.

Training Data

Used to teach the model.

Usually around 80% of the data.

Testing Data

Used after training to check how well the model performs on unseen data.

Usually around 20% of the data.

Real-World Example

Imagine preparing for an exam.

You study from 80 questions.

Then your teacher gives you 20 completely new questions.

If you answer those correctly, it means you really understood the topic.

The testing dataset works exactly like this.

 Data Scaling (Standardization)

The values in a dataset can have very different ranges.

For example:

Study Hours:

1 to 10

Attendance:

50 to 100

Previous Marks:

40 to 100

Large differences in values can confuse the model.

Scaling converts all features to a similar range so the neural network learns more efficiently.

Why is Scaling Important?
Faster learning
Better accuracy
Stable training
Real-World Example

Imagine running a race where one person measures distance in meters and another in kilometers.

It becomes difficult to compare.

Scaling converts everything into the same measurement system.

 Building the Neural Network:

The neural network consists of different layers.

Information flows from one layer to another until a prediction is made.

1. Input Layer

The Input Layer receives all the information.

In this project, the model receives four features:

Study Hours
Sleep Hours
Attendance
Previous Marks

The Input Layer does not make decisions.

It simply passes the data to the next layer.

Real-World Example

Like a student filling out an admission form.

The form collects information before processing it.

2. Hidden Layers

Hidden Layers are where the actual learning happens.

These layers analyze the input data and discover hidden relationships.

The project uses three hidden layers.

Each hidden layer contains several neurons.

More neurons allow the model to learn more complex patterns.

Real-World Example

Imagine three experienced teachers discussing a student's performance.

Teacher 1 checks study habits.

Teacher 2 checks attendance.

Teacher 3 checks previous marks.

Together they decide whether the student will pass.

Hidden layers work similarly.

 What is a Neuron?

A neuron is the smallest learning unit inside a neural network.

Each neuron receives information, processes it, and sends the result to the next neuron.

Real-World Example

Think of each neuron as a teacher.

Each teacher gives their opinion.

The final decision is made after combining all opinions.

 Activation Function (ReLU):

Neurons use activation functions to decide what information should move forward.

This project uses ReLU (Rectified Linear Unit).

ReLU ignores negative values and keeps positive values.

Why use ReLU?

Faster learning
Reduces unnecessary calculations
Helps deep neural networks train efficiently
Real-World Example

Imagine a security guard.

Only authorized people are allowed inside.

Similarly, ReLU allows useful information to continue and blocks unnecessary values.

 Dropout Layer:

Dropout is used to prevent Overfitting.

During training, it randomly turns off some neurons.

This forces the model to learn from different combinations of neurons instead of memorizing the data.

What is Overfitting?

Overfitting happens when a model memorizes the training data instead of learning general patterns.

Such a model performs well on training data but poorly on new data.

Why Use Dropout?
Improves generalization
Prevents memorization
Makes the model more reliable
Real-World Example

Imagine a student memorizing past exam papers.

If the exam changes slightly, the student struggles.

A student who understands the concepts performs well in any exam.

Dropout encourages understanding instead of memorization.

 Output Layer:

The Output Layer gives the final prediction.

Since this project predicts only two classes:

Pass
Fail

The output layer has one neuron.

It uses the Sigmoid Activation Function.

Sigmoid produces a value between 0 and 1.

This value represents the probability.

Example:

0.95 → Very likely to pass

0.20 → Very likely to fail

⚙️ Model Compilation:

Before training begins, the model must be prepared.

Compilation tells TensorFlow:

How to learn
How to measure mistakes
How to evaluate performance

 Optimizer (Adam)

The optimizer updates the neural network's weights after every learning step.

This project uses Adam, one of the most popular optimizers.

Why Adam?

Fast learning
Stable performance
Works well for most AI problems
Real-World Example

Imagine a student solving math problems.

After every mistake, the student adjusts their approach.

Adam helps the neural network improve in the same way.

❌ Loss Function (Binary Crossentropy)

Loss measures how wrong the model's predictions are.

Lower loss means better predictions.

This project uses Binary Crossentropy because there are only two classes:

Pass
Fail
Real-World Example

A teacher compares the student's answer with the correct answer.

The more mistakes, the higher the loss.

📈 Accuracy

Accuracy tells us how many predictions are correct.

Example:

100 predictions

95 correct

Accuracy = 95%

Higher accuracy means better performance.

 Model Training:

Training is the process where the model learns from the training dataset.

During training:

The model sees all training examples.
It finds patterns.
It adjusts its weights.
It improves after every round.

🔄 Epoch:

An Epoch means the model has seen the entire training dataset once.

Example:

50 epochs means the model studies the complete dataset 50 times.

Real-World Example

Reading an entire textbook one time equals one epoch.

Reading it 10 times equals 10 epochs.

More epochs often improve learning, but too many can cause overfitting.

📦 Batch Size

Instead of learning from all data at once, the model learns in smaller groups called batches.

Example:

Batch Size = 32

The model processes 32 students at a time.

Why Use Batches?
Faster training
Less memory usage
More stable learning
Real-World Example

A teacher checks 32 exam papers at a time instead of checking all 1000 together.

 Validation Data:

Validation data is a small portion of the training data used during training.

It helps monitor how well the model performs while learning.

It is not used for learning, only for checking progress.

Real-World Example

A teacher gives weekly quizzes during the semester to check students' understanding before the final exam.

 Model Evaluation:

After training, the model is tested using the testing dataset.

Two important values are measured:

Test Accuracy

Shows how many predictions were correct on unseen data.

Test Loss

Shows how many mistakes the model made on unseen data.

A good model usually has:

High Accuracy
Low Loss
Training History

During training, TensorFlow stores the model's progress.

Two graphs are commonly used:

Accuracy Graph

Shows how prediction accuracy improves over time.

A rising accuracy curve indicates that the model is learning.

Loss Graph

Shows how prediction errors decrease during training.

A falling loss curve indicates better learning.

Real-World Example

Imagine practicing cricket every day.

Runs scored increase over time (Accuracy).

Mistakes decrease over time (Loss).

 Making Predictions:

After training, the model can predict results for new students it has never seen before.

The model receives:

Study Hours

It predicts:

Pass
Fail

It also provides a confidence score (probability).

Real-World Example

 Quick Summary:

A Neural Network is an AI model inspired by the human brain that learns patterns from data.
The model predicts whether a student will Pass or Fail using four input features: Study Hours, Sleep Hours, Attendance, and Previous Marks.
The dataset is split into training data (to teach the model) and testing data (to evaluate its performance).
Data scaling standardizes feature values, helping the model train faster and more accurately.
The neural network consists of:
Input Layer – receives the four input features.
Hidden Layers – learn patterns using neurons and ReLU activation.
Dropout Layers – reduce overfitting by randomly disabling some neurons during training.
Output Layer – uses the Sigmoid activation function to predict the probability of Pass or Fail.
The model is compiled using:
Adam Optimizer – adjusts weights to improve learning.
Binary Crossentropy Loss – measures prediction errors for binary classification.
Accuracy – tracks the percentage of correct predictions.

During training:

An Epoch is one complete pass through the training data.
Batch Size determines how many samples are processed at once.
Validation Data monitors performance during training without updating the model.
After training, the model is evaluated using Test Accuracy and Test Loss.
Finally, the trained model predicts outcomes for new students and provides a confidence score for each prediction.

One-Line Concept:

A neural network learns from past student data, discovers hidden patterns, and predicts whether a new student is likely to pass or fail with a probability score.
