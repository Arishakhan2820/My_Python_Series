📘 TOPIC 4 — Classification (Logistic Regression)

What is Classification?

Classification is a Supervised Machine Learning technique used to predict categories or classes instead of numerical values.

Unlike Linear Regression, which predicts numbers, Classification predicts labels such as Yes/No, Pass/Fail, or Spam/Not Spam.

Simple Definition

Classification is a Machine Learning technique that learns from labeled data and predicts which category a new data point belongs to.

What is Logistic Regression?

Logistic Regression is one of the most popular classification algorithms.

Although its name contains "Regression", it is mainly used for classification problems, especially when there are only two possible outcomes.

For example:

Pass or Fail
Spam or Not Spam
Disease or No Disease
Fraud or Not Fraud
Why Do We Use Logistic Regression?

We use Logistic Regression when we want the model to decide which class an item belongs to instead of predicting an exact number.

How Does Logistic Regression Work?

The model learns patterns from the training data and calculates the probability that an observation belongs to a particular class.

If the probability is greater than 0.5, it predicts Class 1.

If the probability is less than 0.5, it predicts Class 0.

Workflow
Collect data.
Separate Features (X) and Target (y).
Split data into Training and Testing sets.
Scale the features using StandardScaler.
Train the Logistic Regression model.
Predict categories.
Evaluate the model using Accuracy, Classification Report, and Confusion Matrix.
Important Terms
Features (X)

Input values given to the model.

Example:

Study Hours
Attendance
Target (y)

The value the model predicts.

Example:

Pass (1)
Fail (0)
Feature Scaling

Converts different feature ranges into a similar scale for better model performance.

Probability

Logistic Regression predicts the probability of belonging to a class.

Example:

Pass Probability = 95%

Accuracy

Shows how many predictions are correct.

Formula:

Accuracy = Correct Predictions ÷ Total Predictions

Classification Report

Provides detailed evaluation using Precision, Recall, F1-Score, and Support.

Confusion Matrix

A table that compares actual and predicted classes.

Actual	Predicted	Meaning
Fail	Fail	True Negative (Correct)
Pass	Pass	True Positive (Correct)
Fail	Pass	False Positive (Wrong Alarm)
Pass	Fail	False Negative (Missed Prediction)

Real-World Applications

Predicting whether a student will pass or fail.
Detecting spam emails.
Credit card fraud detection.
Disease diagnosis.
Customer churn prediction.
Loan approval systems.
Face recognition.
Fake news detection.
Advantages
Simple and easy to understand.
Fast training.
Works well for binary classification.
Predicts probabilities along with classes.
Easy to interpret.
Limitations
Best suited for linear decision boundaries.
Not ideal for highly complex datasets.
Sensitive to outliers.
Requires proper feature scaling for better performance.
Difference Between Linear Regression and Logistic Regression
Linear Regression	Logistic Regression
Predicts numbers	Predicts categories
Output is continuous	Output is discrete
Example: Predict Marks	Example: Predict Pass/Fail
Evaluation: RMSE, R²	Evaluation: Accuracy, Confusion Matrix
