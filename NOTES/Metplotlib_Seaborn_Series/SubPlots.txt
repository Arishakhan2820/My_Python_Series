📘 Topic 6 — Subplots (Multiple Graphs in One Figure)

What are Subplots?

Subplots allow us to display multiple graphs inside a single figure.

In simple words,

Subplots help us show several charts together in one window, making it easier to compare and analyze different types of data at the same time.

Instead of creating separate windows for each graph, Subplots organize multiple graphs into a grid.

Definition

A Subplot is a smaller graph placed inside a larger figure. It allows multiple plots to be displayed together in a single figure using rows and columns.

Why Do We Use Subplots?

Subplots are useful when we want to:

Compare multiple datasets.
Display different chart types together.
Build professional dashboards.
Save space by combining graphs.
Analyze data more efficiently.

Instead of opening many graph windows, everything appears in one organized figure.

What is a Figure?

A Figure is the complete canvas or window that contains one or more graphs.

Think of it as a page.

Inside this page, we place one or more subplots.

What is an Axis (Axes)?

Each individual graph inside a figure is called an Axes (or subplot).

Every subplot has its own:

X-axis
Y-axis
Title
Labels
Grid
Data

This means each graph works independently, even though they are displayed together.

Grid Layout

Subplots are arranged using rows and columns.

For example:

1 × 2 Grid
+----------------+----------------+
|    Plot 1      |    Plot 2      |
+----------------+----------------+
