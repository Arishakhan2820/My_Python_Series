📘 Topic 4 — Scatter Plot (Relationship Between Variables)

What is a Scatter Plot?

A Scatter Plot is a graph that shows the relationship between two numerical variables using individual points.

In simple words,

A Scatter Plot helps us understand whether two variables are related and how one variable changes when the other changes.

Each point on the graph represents one observation (or one data record).

Definition

A Scatter Plot is a graphical representation of two numerical variables where each data point is plotted according to its x-coordinate and y-coordinate. It is mainly used to identify relationships, patterns, trends, and correlations between variables.

Why Do We Use a Scatter Plot?

A Scatter Plot helps answer questions like:

Does one variable affect another?
Is there a relationship between two variables?
Do values increase together?
Do values decrease together?
Are there any unusual data points (outliers)?

Instead of reading numbers, a Scatter Plot makes relationships easy to understand.

What Does a Scatter Plot Show?

A Scatter Plot shows:

Relationship between variables.
Positive correlation.
Negative correlation.
No correlation.
Trends.
Outliers.
Clusters of data.
Components of a Scatter Plot

A Scatter Plot usually contains:

X-axis (Independent Variable)
Y-axis (Dependent Variable)
Data Points
Title
Axis Labels
Grid Lines
Trend Line (optional)
Color Scale (optional)
Legend (optional)
What is Correlation?

Correlation means how two variables are related to each other.

There are three main types.

1. Positive Correlation 📈

When one variable increases, the other also increases.

Example:

More study hours → Higher marks
More experience → Higher salary
More exercise → Better fitness

Points move upward from left to right.

2. Negative Correlation 📉

When one variable increases, the other decreases.

Example:

Higher product price → Lower sales
More screen time → Less sleep
Faster speed → Less travel time

Points move downward from left to right.

3. No Correlation

The variables have no clear relationship.

Example:

Shoe size vs Exam marks
Favorite color vs Salary

The points appear randomly scattered.

What is a Trend Line?

A Trend Line (also called a Best Fit Line) is a straight line that shows the overall direction of the data.

It helps us understand whether the relationship is:

Increasing
Decreasing
Nearly constant

The trend line summarizes the overall pattern of the data.

Why Use Colors?

Colors help represent additional information.

For example:

Higher marks → Green
Average marks → Yellow
Lower marks → Red

This makes the graph easier to understand.

Why Use Different Point Sizes?

Point size can represent another variable.

For example:

Small point → Low sales
Large point → High sales

This allows one graph to display multiple pieces of information.

🌍 Real-World Example 1 — Study Hours vs Marks

A teacher wants to know:

Does studying more improve exam marks?

A Scatter Plot is created:

X-axis → Study Hours
Y-axis → Marks

The graph shows:

Students who study more usually score higher marks.
A positive trend is visible.

🌍 Real-World Example 2 — Advertising vs Sales

A company spends money on advertising.

They create a Scatter Plot:

X-axis → Advertising Budget
Y-axis → Sales

If the points move upward, it means:

More advertising leads to higher sales.

🌍 Real-World Example 3 — Exercise vs Weight Loss

A fitness trainer records:

Exercise hours
Weight loss

A Scatter Plot helps determine whether exercising more leads to greater weight loss.

🌍 Real-World Example 4 — House Size vs Price

A real estate company compares:

House Size
House Price

The Scatter Plot usually shows:

Larger houses generally cost more.

This indicates a positive relationship.

Why is Scatter Plot Important in AI & Machine Learning?

Before training Machine Learning models, Data Scientists use Scatter Plots to:

Study relationships between features.
Detect outliers.
Identify trends.
Check linear relationships.
Understand feature behavior.
Select useful features for models.

Scatter Plots are an important part of Exploratory Data Analysis (EDA).

Advantages of Scatter Plots
Easy to understand relationships.
Detects positive and negative correlation.
Finds trends.
Identifies outliers.
Helps build better Machine Learning models.
Supports predictive analysis.
Works well with numerical data.
Limitations of Scatter Plots
Only suitable for numerical variables.
Large datasets may become crowded.
Does not show exact frequency.
Less useful for categorical data.

Scatter Plot vs Bar Chart vs Histogram
Scatter Plot	Bar Chart	Histogram
Shows relationship between two numerical variables	Compares categories	Shows distribution of numerical data
Uses individual points	Uses bars	Uses touching bars (bins)
Best for correlation	Best for comparison	Best for distribution
